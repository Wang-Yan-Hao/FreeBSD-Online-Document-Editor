= FreeBSD 社区

有超过一百个 link:https://www.FreeBSD.org/community/mailinglists/[邮件列表]， 许多网页 https://forums.FreeBSD.org/[论坛]， 以及几个 link:{handbook}#eresources-news[新闻组] 可供您查看。 在世界上 {{< get-usergroups-info "countries" >}} 个不同的国家有超过 个 {{< get-usergroups-info "usergroups" >}} link:https://www.FreeBSD.org/usergroups/[用户群组]， 和一个活跃的 link:https://www.FreeBSD.org/community/irc/[IRC] 社区。 很多开发者维护了各种 https://planet.freebsd.org[博客] 来展示他们为 FreeBSD 所做的工作。 开发者和关键的贡献者也维护了一个 https://wiki.FreeBSD.org/[wiki]， 其中包括了 FreeBSD 开发进度和相关项目的信息。

过去的一年， 在世界上 {{< get-event-last-year-info "countries" >}} 个不同的国家举办了 {{< get-event-last-year-info "events" >}} 场 FreeBSD 活动。 若要了解未来与 FreeBSD 有关的活动， 请查阅 link:https://www.FreeBSD.org/events/[活动页面] 上维护的 日历 和 link:https://www.FreeBSD.org/events/feed.xml[RSS 源]。 在 YouTube 上的 https://www.youtube.com/bsdconferences[BSD 会议] 频道里有许多过去活动的视频。

== 社交网络

FreeBSD 出现在许多不同的社交网络上。

* 在 https://www.flickr.com[flickr] 上， 有上千张来自用户群组， 会议和黑客马拉松的照片被加上了 'link:https://flickr.com/search/?z=t&ss=2&w=all&q=freebsd&m=text[freebsd]' 标签。
* 在 https://www.youtube.com[YouTube] 上有上百段会议视频， 屏幕录像， 以及关于 https://www.youtube.com/results?search_query=freebsd[FreeBSD] 的演示。 特别要提到的是， 有一个新的 https://www.youtube.com/bsdconferences[BSD 会议] 频道， 其中包含了来自 FreeBSD 技术会议的各种一小时 FreeBSD 录制演讲。
* 在 https://www.facebook.com[Facebook] 上有一个 https://www.facebook.com/groups/FreeSBD/[FreeBSD 用户群组]， 在link:https://www.linkedin.com[LinkedIn] 上有一个 link:https://www.linkedin.com/groups?gid=47628[FreeBSD 群组]。
* 您可以在 https://twitter.com[Twitter] 上关注 https://twitter.com/freebsd[@freebsd]， https://twitter.com/freebsdhelp[@freebsdhelp]， https://twitter.com/freebsdblogs[@freebsdblogs]， 以及 https://twitter.com/freebsdcore[@freebsdcore]。
