= FreeBSD/i386 Project

[[intro]]
== 介绍

本页包含的信息是关于 FreeBSD/i386 平台的。绝大部分的家庭用户购买 的计算机的 CPU 都是由这个架构支持的。如果您拥有一台任何一家大型 PC 厂商的计算机 (例如戴尔(TM)，HP 或 eMachines(R))， 您的计算机很有可能拥有一颗 AMD(TM) 或者 Intel(R) 的 CPU，它们都是属于这个类别的。

[[news]]
== 最新消息

* FreeBSD 6.0 及后继版本已经不再支持 Intel(TM) 80386 CPU； 这个 CPU 已经停产超过七年了，所以只有 FreeBSD 5.X 以及更早以前的版本支持。需要注意的是只有 FreeBSD 4.X 以及以前的版本的 GENERIC 内核才支持 80386 平台。
