= 联系 FreeBSD

== 关于 FreeBSD 的问题...

关于 FreeBSD 的问题应该发送到 FreeBSD Questions 邮件列表， freebsd-questions@FreeBSD.org。

link:../../community/mailinglists/[邮件列表] 是提供给 FreeBSD 用户的主要支持渠道， 包含了众多不同主题的邮件列表。 同时，也有许多非英语的邮件列表。

== 关于此 WWW 服务器的内容...

关于我们的文档 (link:{handbook}[手册], link:{faq}[FAQ], link:../docs/[书籍和文章]) 的问题应当发给 FreeBSD Documentation Project 邮件列表， freebsd-doc@FreeBSD.org。
