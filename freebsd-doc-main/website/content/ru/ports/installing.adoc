= Установка Коллекции портов FreeBSD

{{< form-ports >}}

'''''

Программа установки FreeBSD позволяет установить Коллекцию Портов вместе с установкой операционной системы. Обратитесь к странице Справочника link:https://www.FreeBSD.org/cgi/man.cgi?query=bsdinstall[`bsdinstall`] для FreeBSD 9.X и более новых версий, или link:https://www.FreeBSD.org/cgi/man.cgi?query=sysinstall[`sysinstall`] для FreeBSD 8.X и более старых версий.

Если Коллекция Портов изначально не установлена, вы можете её добавить с помощью link:{handbook}#ports-using[`portsnap`] или link:{handbook}#svn[Subversion].
