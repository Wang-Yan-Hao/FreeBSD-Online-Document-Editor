Help Page - FAQ

Q: What is FreeBSD Online Document Editor?
A: It is an online website editor for FreeBSD documentation which is still developing. The purpose of FreeBSD Online Document Editor is to simplify the editing process in FreeBSD documentation. Here is the GitHub repo URL - https://github.com/Wang-Yan-Hao/FreeBSD-Online-Document-Editor.
Q: How to use the online editor to change document>
A: Please follow the below step.
  1. You find the problem on a certain page on the FreeBSD document.
  2. Click the "Edit Page" button on the page and go to "FreeBSD Online Document Editor".
  3. Change the source code(.adoc) in the left section. Click the "Patch Download" button to download the patch about your change.
  4. Click the "FreeBSD Bugzilla" button, going to FreeBSD Bugzilla to submit the patch.
